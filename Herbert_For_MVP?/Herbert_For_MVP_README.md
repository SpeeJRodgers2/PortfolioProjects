# Did Justin Herbert Deserve an MVP Vote in 2025?
### By: Spencer Rodgers
### 2/19/26

I wrote this paper to spite all of my friends that roasted me in the group chat for saying Sam Monson’s MVP vote for Justin Herbert is justifiable and not egregiously bad like most people are acting. Many people are arguing that Drake Maye should have won the MVP over Matthew Stafford this year. Some are even blaming Monsen for the result because a 1st place vote for Maye from him could’ve actually swung the MVP race. This anger towards Sam Monsen and the roast of him, and now myself, is unjust! Now, let’s get into why.
